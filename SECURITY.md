Security Policy
Supported Version

This project is actively maintained, and only the latest version of Mafia RPG is supported with security and bug fixes.

Version	Supported
Latest	Yes

Older versions are not guaranteed to receive updates or security fixes.

Reporting a Vulnerability

If you discover a security vulnerability, exploit, or unintended behavior, please report it responsibly.

You can report issues by:

Opening a GitHub Issue in this repository
Contacting the project maintainer directly (if contact information is provided)

When reporting, please include:

A clear description of the issue
Steps to reproduce the problem
Any relevant screenshots, code snippets, or examples
Response Process
Reports will be reviewed as soon as possible
Critical vulnerabilities will be prioritized
You will receive a response indicating whether the issue has been accepted, investigated, or resolved
Responsible Disclosure

Do not publicly disclose or exploit vulnerabilities before they have been reviewed and addressed. Responsible reporting helps maintain the stability and integrity of the project.
